What’s Annotaion

There are three built-in annotations available in Java:

  • Override: We use Override to mark methods which overriding a method
  • Deprecated: The marked element(class, methods or filed) is deprecated and should no longer be used. The cimplier generates a warning whenever a program uses a deprecated element.
  • SuppresWarnings: When “deprecation” was passed to @SuppressWarnings(“deprecation”), deprecation warnings would be suppressed.

Those annotations can be used for giving certain instuction to compiler.

**Annotations can be also used as Compile-time instructors: ** This can be further used by software build tools for generating code, XML etc.

Runtim instructions: Be used to give instructions to the program at runtime.

Custom Annotations

  • Custom Annotations can be created by using @interface.
  • An Annotation can have elements
  • Alll Annnotations extends java.lang.annotation.Annotation interface. Annotations cannots include any extends clause.
