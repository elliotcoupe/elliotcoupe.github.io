Get Started with Hexo Blog-generator and deploy
Elliot Coupe Lv1
 2021-01-27 00:18:17

Hexo is a easy generator of personal blog based on Node.js

With few simple step you can have your personal block generated.
Moreover, the following generation of new blogs can be easiy done with CLI.

Start with Using Hexo

  • Dowload Node js
  • Install Nexo cli
  • Nexo init
  • instll NPM packages
  • Start bloging on local with: nexo s
  • Generate new blogs with: hexo new “”
  • Use: hexo g to regenerate the static files

To deploy the blog with hexo-deployer-git

  • NPM install neo-deployer-git
  • Modify the _configure.yml to add type:git repo: repo and branch:master
  • Deploy with hexo d
