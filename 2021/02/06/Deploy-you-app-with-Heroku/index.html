Deploy your app with Heroku
Elliot Coupe Lv2
 2021-02-06 23:51:18

Heroku is a cloud platform as service supporting several programming languages. Free tier user will be able to deploy there app for free with Hereoku. More importantly, Heroku provdes a great deployment experience which is really fuzz-free for new users.

Heroku CLI

After registeration you will need to download Heroku CLI whihc provide better experience in log checking.

A Little bit modify on Node project

Cd into your project and make sure this project has a Github repository.

Add the start script to Package.json, which will be used by Heroku to start the APP

Heroku dynos expose a dynamic port for your app to bind to. This value is exposed in the $PORT env var.

Deployment

Create an app on Heroku. Heroku will create a random name for your app

Deploy with

Use Heroku ps:scale to open on web instance.

Then you deployrment with: heroku open
