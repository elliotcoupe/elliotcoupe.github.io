In the runtme of Javascript, there is a Stack and a Queue. Stack holds the state of the fnction is currently running, and the Queue is the wating area for functions. Wheneve the stack is empty, the event loop will check the queue for any waiting messages and add to the stack.

In the example, the setTimeout was first put into the stack and it triggered a timer and adder added the console.log(1) to the Queue, when console.log(2) was executed, the event loop bring console.log(1) from the Queue to the Stack.

Callback

Callback functions are function been passed to a another function as a argument, which are not asynchronous by nature but can be used for asynchrous purple.

Take setTimeout() as an example, the first paramenter is the callback function which allow you to be informed of when as asychronous task was finished and also be able deal with the result.
